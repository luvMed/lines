How to Use Generated CSS

  1. Upload an image in the main application
  2. Adjust the settings to your preference
  3. Generate the mathematical outlines
  4. Copy the generated CSS code
  5. Paste it into your website's stylesheet
  6. Add the corresponding HTML structure
/* Example of generated CSS from the main application */ .math-outline { position: relative; width: 100%; height: 100%; } .outline-0 { position: absolute; pointer-events: none; } .outline-0 path { fill: none; stroke: #667eea; stroke-width: 2; stroke-linecap: round; stroke-linejoin: round; } .outline-0 svg { width: 100%; height: 100%; }
